# ☕ Бот для предзаказа кофе

Полнофункциональный Telegram-бот для предзаказа кофе с системой бонусов, администрированием и массовой рассылкой.

## 🚀 Возможности

### Для клиентов:
- ✅ Регистрация с указанием ФИО, телефона и даты рождения
- 📋 Просмотр меню кофе с ценами
- 🛒 Оформление предзаказа с выбором размера, молока и сиропа
- 💰 Система бонусов (начисление/списание)
- 🎁 Кэшбэк с каждого заказа
- 🎂 Бонусы на день рождения (500 бонусов)
- 📱 Уведомления о статусе заказа
- 📋 История заказов

### Для администраторов:
- 👨‍💼 Система смен ("На смене"/"Сойти со смены")
- 📦 Управление заказами:
  - Принять заказ
  - Установить время приготовления
  - Отметить заказ готовым
  - Отменить заказ
- 📊 Просмотр активных заказов
- 📢 Массовая рассылка с изображениями
- 👤 Информация о клиенте при заказе

## 🛠 Установка и запуск

### 1. Получение токена бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям, задайте имя и username бота
4. Скопируйте полученный токен (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2. Настройка конфигурации

1. Переименуйте файл `.env.example` в `.env`:
   ```bash
   mv .env.example .env
   ```

2. Откройте файл `.env` и заполните:
   ```env
   BOT_TOKEN=your_bot_token_here
   ADMIN_IDS=123456789,987654321,456789123
   ADMIN_NAMES=Денис,Влада,Настя
   COFFEE_SHOP_NAME=Моя Кофейня
   WELCOME_BONUS=100
   BIRTHDAY_BONUS=500
   CASHBACK_PERCENT=5
   ```

   **Как узнать свой Telegram ID:**
   - Напишите [@userinfobot](https://t.me/userinfobot) в Telegram
   - Или [@getidsbot](https://t.me/getidsbot)

### 3. Размещение на bothost.ru

#### Шаг 1: Регистрация
1. Перейдите на [bothost.ru](https://bothost.ru)
2. Зарегистрируйтесь или войдите в аккаунт

#### Шаг 2: Создание бота
1. Нажмите "Создать бот" или "Добавить бот"
2. Введите название для вашего бота
3. Выберите тип: **Python**

#### Шаг 3: Загрузка файлов
1. В панели управления найдите раздел "Файлы" или "Редактор"
2. Загрузите все файлы проекта:
   - `bot.py`
   - `config.py`
   - `database.py`
   - `requirements.txt`
   - `.env`

#### Шаг 4: Установка зависимостей
В разделе "Зависимости" или "Requirements" убедитесь, что установлены:
```
python-telegram-bot==20.7
python-telegram-bot[job-queue]==20.7
SQLAlchemy==2.0.23
aiosqlite==0.19.0
python-dotenv==1.0.0
aiohttp==3.9.1
pillow==10.1.0
```

#### Шаг 5: Запуск
1. Нажмите кнопку "Запустить" или "Старт"
2. Дождитесь сообщения о успешном запуске
3. Проверьте работу бота в Telegram

## 🎯 Использование бота

### Для клиентов:
1. Найдите бота в Telegram по username
2. Нажмите **Start** или отправьте `/start`
3. Пройдите регистрацию (ФИО, телефон, дата рождения)
4. Используйте кнопки меню:
   - ☕ **Меню** - посмотреть и заказать кофе
   - 💰 **Бонусы** - проверить баланс
   - 📋 **История заказов** - посмотреть прошлые заказы
   - 🛒 **Мой заказ** - статус текущего заказа

### Для администраторов:
1. Нажмите **👨‍💼 Панель администратора**
2. Нажмите **✅ На смене** чтобы начать принимать заказы
3. При поступлении заказа вы получите уведомление с кнопками:
   - **✅ Принять заказ** - принять заказ
   - **❌ Отменить заказ** - отменить заказ
4. После принятия выберите время приготовления
5. Когда заказ готов, нажмите **✅ Заказ готов**

### Массовая рассылка:
1. В админ-панели нажмите **📢 Массовая рассылка**
2. Введите текст сообщения
3. Отправьте изображение (опционально) или нажмите "Пропустить"
4. Сообщение будет отправлено всем зарегистрированным пользователям

## 🔧 Техническая информация

### База данных
Бот использует SQLite для хранения:
- Пользователей (id, ФИО, телефон, дата рождения, бонусы)
- Администраторов (id, имя, статус смены)
- Заказов (детали заказа, статус, время)
- Рассылок (история массовых рассылок)

### Статусы заказов:
- `pending` - Ожидает принятия
- `accepted` - Принят администратором
- `preparing` - Готовится
- `ready` - Готов к выдаче
- `completed` - Выполнен
- `cancelled` - Отменен

### Уведомления клиентам:
- ✅ "Ваш заказ принят"
- ⏱️ "Заказ готовится" + время
- ✨ "Заказ готов"
- ❌ "Заказ отменен"

## 🐛 Устранение неполадок

### Бот не отвечает:
1. Проверьте правильность токена в `.env`
2. Убедитесь, что бот запущен в панели bothost.ru
3. Проверьте логи в панели управления

### Ошибка базы данных:
1. Перезапустите бота
2. При необходимости удалите файл `coffee_bot.db` (будет создан заново)

### Не приходят уведомления админу:
1. Проверьте правильность ADMIN_IDS в `.env`
2. Убедитесь, что администратор нажал "На смене"
3. Проверьте, что администратор запустил бота в Telegram
